flex - Overview

The flex gem provides the core infrastructure and the lower level functionality used also by all the other flex gems:

  • HTTP clients (see Configuration)
  • elasticsearch API methods (see API Methods)
  • Templating System (see Templating)
  • Cascading Variables Management (see Variables)
  • Result Extenders (see Result Extenders)
  • Logging & Debugging (see Configuration)
  • Self-documenting tool (see Self Documentation)
  • Rake Tasks (see Rake Tasks)

This section contains the specific documentation of the gem, if you want a general overview of the flex project you should read the Project Documentation first (see Flex Project)
